# West Harbor Painting Project Rules

## Purpose
Build a production-quality lead-generation website for West Harbor Painting, a residential and commercial painting company serving Los Angeles and Orange County.

## Development Principles
- Build reusable React components.
- Keep components small and focused.
- Prefer server components unless client components are required.
- Use TypeScript, Tailwind CSS, shadcn/ui, and Lucide icons.
- Prioritize accessibility, mobile responsiveness, SEO, and fast loading.
- Do not modify unrelated files.

## Design Direction
The website should feel modern, premium, clean, trustworthy, and contractor-focused.

Color direction:
- Deep navy
- White
- Warm orange accent
- Slate neutrals

## Primary Goal
Increase estimate requests and phone calls.

Every section should support lead generation.

## Future Features
Keep the architecture ready for:
- AI customer assistant
- Estimate request wizard
- Google Analytics
- Google Maps
- Contact form backend
- CRM integration
